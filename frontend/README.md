# Heimdall Frontend

This directory contains the Heimdall marketing landing page built with [Vite](https://vitejs.dev/) and React.

## Getting Started

```bash
npm install
npm run dev
```

The development server starts on http://localhost:3000.

## Building for Production

```bash
npm run build
```

The static output is written to `frontend/build`. Preview the production build locally with:

```bash
npm run preview
```
